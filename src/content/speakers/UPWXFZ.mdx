Sebastiaan is a Senior Software Engineer for the [Ordina Pythoneers](https://ordina.com) and a Fellow of the [Python Software Foundation](https://www.python.org/psf-landing/). He has a passion for teaching Python and writing clean code. Sebastiaan is also active in the Python community, as the Financial Aid lead for EuroPython, open-source contributor, Python educator, and founder of [Python Discord](https://www.pythondiscord.com/).

As a public speaker, Sebastiaan makes frequent appearances at Python conferences, including PyCon US, EuroPython, PyCon APAC, FOSDEM, and Pyjamas Conf. He currently lives in The Netherlands.
